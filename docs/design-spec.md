# Дизайн-спецификация

Бриф: тёмный, премиальный, «продающий безопасность» сайт. Аудитория — частные клиенты
(личная безопасность, бытовые конфликты) и юрлица (объекты, мероприятия). Задача страницы —
вызвать доверие и увести в заявку. Запрет на ИИ-штампы — жёсткий.

## Чего НЕ делаем (анти-паттерны)

- Дефолтные icon-паки (Font Awesome, Material, Lucide-как-есть) — рисуем свой набор.
- Generic «near-black + один кислотный неон-акцент» (это сам по себе ИИ-дефолт). Тёмная база —
  да (по брифу), но акцент сдержанный, тёплый, не неон.
- Glassmorphism, декоративные градиенты, «парящие карточки» с большими размытыми тенями.
- Шрифты Inter / Roboto / Open Sans как основные.
- Стоковая «безопасность» (замки, щиты-клипарт). Только реальные фото в едином грейде.

## Дизайн-направление

Холодная, дисциплинированная, «оперативная» эстетика: матовый графит, точная сетка,
сильная типографика, реальная съёмка в едином тёмном грейде. Премиум читается не золотом
повсюду, а сдержанностью и точностью. Один тёплый акцент — латунь — только на ключевых
действиях и на сигнатурном элементе.

**Сигнатурный элемент:** «периметр» — тонкая рамка-обрамление с угловыми засечками
(как защищённая зона / визир), плюс моноширинные технические подписи (код города, координаты,
тайм-код). Обрамляет герой и ключевые блоки. Это единственная «смелость» — всё остальное тихое.

## Токены

Цвет (тёмная база, холодная; тёплый акцент сдержанно):
```
--bg-900:   #0B0D10   /* матовый почти-чёрный, холодный — фон */
--bg-800:   #12151A   /* поверхность */
--bg-700:   #1A1E25   /* приподнятая поверхность */
--line:     rgba(255,255,255,0.08)   /* хайрлайны */
--line-2:   rgba(255,255,255,0.14)
--text-100: #ECEEF1   /* основной текст, тёплый-нейтральный белый */
--text-300: #9BA1A9   /* приглушённый */
--text-500: #5C626B   /* подписи/мета */
--accent:   #C39A55   /* латунь — ТОЛЬКО CTA и сигнатура, не заливки */
--accent-d: #8A6B34   /* латунь тёмная (бордеры/hover) */
--danger:   #C2503C   /* тревога/ошибки, очень дозированно */
```
Контраст текста к фону — минимум AA (проверять). Светлой темы нет (бренд тёмный).

Типографика — ФИНАЛ (все три — Google Fonts, кириллица есть):
```
--font-display: 'Russo One', sans-serif;    /* Слоган героя + крупные H1. Один вес (400),
                                               тяжёлый техно-гротеск. Грузить только нужные начертания. */
--font-body:    'Manrope', sans-serif;       /* ВЫБРАНО. Весь текст, H2/H3, навигация, кнопки.
                                               Веса 400 / 500 / 700. */
--font-mono:    'Ubuntu Mono', monospace;    /* ВЫБРАНО. Только мелкие тех-подписи, коды городов,
                                               координаты сигнатуры. */
```
Загрузка (subset cyrillic, display=swap): Russo One 400; Manrope 400/500/700; Ubuntu Mono 400.

Правило для Ubuntu Mono: гарнитура мягкая/гуманистическая — чтобы читалась технично,
применять ТОЛЬКО на мелких подписях и координатах, КАПСОМ и с трекингом (`letter-spacing
~0.04–0.14em`). Не использовать в основном тексте и заголовках.

Правила для Russo One (критично — иначе удешевляет макет):
- Применять ДОЗИРОВАННО: только слоган героя и крупные H1. НЕ использовать на H2/H3,
  подзаголовках, длинных строках, в навигации и кнопках — там работает Manrope (medium).
- Вес один (400) — иерархию заголовков строить РАЗМЕРОМ, ТРЕКИНГОМ и ЦВЕТОМ, не весом.
- Каплевидной подачи избегать: крупно, с воздухом, сдержанно по количеству. Чем реже
  появляется, тем дороже читается. Вокруг — тихий нейтральный текст.
- Капс для слогана `МЫ ЗАБОТИМСЯ О ВАШЕЙ БЕЗОПАСНОСТИ` — да; для остальных заголовков
  Russo One предпочтительно sentence case (он и так визуально «громкий»).

Шкала (desktop): display 52/36 (только Russo One, крупно), H2/H3 — Manrope-medium 24/19,
body 17 (line-height 1.6), mono-подписи 13. H2/H3 НЕ ставить Russo One.
Заголовки уровня H2 и ниже — sentence case строго.

Сетка и форма:
- 12-колоночная сетка, широкие поля, много воздуха. Контент-ширина ~1200px.
- `--radius: 4px` (сдержанно; премиум-security не любит «пухлые» скругления). Рамка-периметр —
  без скруглений, угловые засечки.
- Тени почти нет; глубина — через слои `--bg-*` и хайрлайны, не через blur-тени.

Движение (дозированно, чтобы не выдать «ИИ-генерёжку»):
- Один оркестрованный момент на загрузке героя (проявление рамки-периметра + кадра).
- Hover-микровзаимодействия на карточках и CTA (сдвиг/подсветка границы), без излишеств.
- `prefers-reduced-motion` уважается всегда.

## Компоненты (визуальные требования)

- **Hero:** тёмный реальный видео/фото-кадр под градиент-затемнением (CSS-затемнение слоем,
  не декоративный градиент-«акцент»), слоган капсом display-шрифтом, рамка-периметр с
  мини-подписями (город/тайм-код моно). Снизу — ряд из 5 интент-карточек.
- **IntentCard:** строгая карточка с тонкой границей, заголовок направления, короткий
  подзаголовок, hover — подсветка границы латунью + сдвиг стрелки. Без иконок-клипарта;
  допустим кастомный мини-знак.
- **ServiceBlock / CityServiceTemplate:** Answer-First врезка сверху (визуально выделена
  левой латунной чертой, без скругления), далее секции с хайрлайн-разделителями.
- **CTA-кнопки:** основная — латунный контур/заливка-минимум, текст-действие; вторичная —
  призрачная с границей `--line-2`. Telegram-кнопка — та же система, без брендовых градиентов.
- **FAQ:** аккордеон на нативном `<details>`, хайрлайны, моно-нумерация если уместно.
- **Footer:** реквизиты-слот (за флагом), контакты, города списком (перелинковка), мини-карта
  географии.

## Иконки

Свой набор `src/components/icons/` — однообводочные SVG, единая толщина штриха (1.5px),
квадратная сетка 24px, «технический» характер под сигнатуру. Минимально необходимый набор
(охрана, переезд, угроза, соседи, мероприятие, телефон, telegram, стрелка, чек). НЕ тянуть
icon-пак. Декоративные иконки — `aria-hidden`, в кнопках без текста — `aria-label`.

## Работа с медиа (по разбору архива)

- **Единый грейд:** все фото привести к одному тёмному холодному тону (десатурация,
  холодные тени, лёгкий контраст), иначе «сборка с телефона» бросается в глаза.
- **Лица клиентов — убрать/заблюрить полностью.** Сотрудников — со спины/в профиль/силуэтом
  (дискретность + премиальнее выглядит).
- **Оружие — нельзя.** ИИ-видео с пистолетами (Hailuo, ролик №3) НЕ использовать вообще.
  Из ИИ-роликов в запасе максимум 1–2 (за рулём / кортеж) как тёмный ambient-луп с затемнением,
  но приоритет — реальные кадры. Герой — из реальных (Котельническая на закате, Москва-Сити,
  кортеж на трассе), в общем грейде.
- **Дубликаты** (Газпром-Арена ×3 и т.п.) — оставить по одному лучшему.
- **Локальные фото** раскладываются на гео-страницы соответствующих городов (Екб, Москва,
  СПб есть; Пермь/Уфа/Челябинск/Тюмень/Новосибирск — фактуры нет, локальный блок без фейка).
- **3 «бытовые» карточки** (переезд/угрозы/соседи) — специфичного фото нет: решаем атмосферой
  и графикой (кадр + рамка-периметр + типографика), без стоковых клише.
- Все изображения — через `astro:assets`, `webp/avif`, размеры заданы (нулевой CLS),
  ниже первого экрана — lazy.

## Письмо (копирайт в интерфейсе)

Спокойно, конкретно, без запугивания и пафоса. Активный залог, плоские глаголы, sentence case.
Кнопка называет действие и сохраняет имя по всему флоу («Оставить заявку» → тост «Заявка
отправлена»). Пустые состояния и ошибки — голосом интерфейса, по делу, без извинений-воды.

## Качество (флор, не объявляем его в UI)

Адаптив до мобильного, видимый фокус клавиатуры, reduced-motion, контраст AA, нулевой CLS,
семантический HTML. Сборка проверяется скриншотами на ключевых разрешениях.
